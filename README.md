# HVC2_Vue_Frontend2025

โปรเจคนี้เป็นส่วนหนึ่งของวิชาการพัฒนาซอฟต์แวร์ด้วยเทคโนโลยี Back-End
ซึ่งใช้ Node.js และ Express.js ในการสร้าง API สำหรับการจัดการข้อมูลผู้ใช้และรถ
ซึ่งใช้ MongoDB ในการจัดการฐานข้อมูล
ใช้ Mongoose ในการสร้าง Schema และ Model
ใช้ dotenv ในการจัดการ environment variables
ใช้ bcryptjs ในการเข้ารหัส password
ใช้ jsonwebtoken ในการสร้าง token
ใช้ cors ในการจัดการ Cross-Origin Resource Sharing
